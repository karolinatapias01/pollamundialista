# ⚽ PollaMundialista 2026

Aplicación web para hacer pronósticos del Mundial de Fútbol 2026 con amigos y familia.

## 🎯 Características

- ✓ Registro simple con código de acceso
- ⚽ Sistema de pronósticos para todos los partidos
- 🏆 Ranking en tiempo real
- 🎯 Sistema de puntos con bonuses
- 🏅 Badges y logros desbloqueables
- 😊 Reacciones en partidos finalizados
- 👑 Panel de administración
- 📱 Diseño responsive para móviles

## 🎮 Sistema de Puntos

### Fase de Grupos y Eliminatorias

- **+2 pts** por acertar el resultado (gana/empata/pierde)
- **+5 pts adicionales** por acertar el marcador exacto (total 7 pts)
- **+3 pts bonus** por racha de 3 aciertos consecutivos
- **+8 pts** por acertar al campeón del mundial

## 🏅 Logros

- 🎯 **Francotirador** - 3 marcadores exactos
- 🔥 **En Llamas** - Racha de 5 aciertos
- 👑 **Oráculo** - Acertar al campeón
- 🔮 **Profeta** - Más del 70% de aciertos
- · **Consistente** - Pronosticar todos los partidos de grupos
- 🍀 **Suertudo** - Acertar una sorpresa
- 🎖️ **Participante** - Primer pronóstico
- 💪 **Dedicado** - Más de 30 pronósticos
